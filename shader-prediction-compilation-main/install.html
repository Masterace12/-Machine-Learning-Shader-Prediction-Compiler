🚀 Quick Install

Copy and paste this command into Konsole:

curl -sL https://raw.githubusercontent.com/Masterace12/shader-prediction-compilation/main/web-install.sh | bash
That's it! The installer will automatically:
  • Download the latest version from GitHub
  • Fix all compatibility issues
  • Install dependencies
  • Configure for your Steam Deck model
  • Set up Steam integration

Step 1: Open Konsole

In Desktop Mode: Applications → System → Konsole
Or press Ctrl+Alt+T

Step 2: Paste Command

Copy the command above and paste it into Konsole, then press Enter.

Step 3: Wait for Completion

The installer will run automatically. You'll see progress messages.

Step 4: Enjoy!

Find the app in:
• Gaming Mode: Library → Non-Steam → Shader Predictive Compiler
• Desktop Mode: Applications → Games → Shader Predictive Compiler

🔄 Alternative Methods

If the main command doesn't work, try these:

Git Clone Method:

git clone https://github.com/Masterace12/shader-prediction-compilation.git /tmp/shader && cd /tmp/shader && bash INSTALL.sh

Wget Method:

wget -qO- https://raw.githubusercontent.com/Masterace12/shader-prediction-compilation/main/web-install.sh | bash
⚠️ Security Note: These commands download and execute code from GitHub. Only run commands from trusted sources. You can always inspect the code first by visiting the GitHub repository.

🗑️ Uninstall

To remove the application:

/opt/shader-predict-compile/install --uninstall
